# polyCID

polyCID is an user friendly R app based on preprocessed GBS data for detect and classify putative contaminants in biparental polyploid population, including apomictic clones, self-fertilization, half-siblings and/or full contaminants individuals. You can explore our simulated data to learn how to interact with the tool, as well as access, modify all datasets and adapt them to your research needs. By going through the workflow below, you will become familiar with all the steps involved in the analysis. Enjoy it!
